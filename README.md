# 📝 Notepad Project

A simple **Notepad application** built with modern web technologies.  
This project provides basic text editing functionalities along with theme customization.

---

## ✨ Features
- ✅ Write and edit text  
- ✅ Undo / Redo  
- ✅ Cut / Copy / Paste  
- ✅ Select All  
- ✅ Dark Theme & Light Theme Toggle
